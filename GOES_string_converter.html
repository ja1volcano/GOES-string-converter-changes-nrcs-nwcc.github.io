FP2 GOES String Converter

This utility converts a GOES string to engineering units. The string can contain the DCS appeneded parentheses at the beginning; the script ignores them. Extra spaces before or after the string must be removed. If a GOES mapping string and header information is provided the utility will attempt to seperate the values into the channels and groups outlined by the mapping string.
